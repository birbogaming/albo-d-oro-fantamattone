# Cartella Classifiche

Questa cartella contiene le immagini delle classifiche finali delle competizioni del Fantamattone.

## Convenzione di Nomenclatura

I file devono seguire il formato: `{STAGIONE}_{TIPO_COMPETIZIONE}.{ESTENSIONE}`

### Formato Stagione
- Utilizzare trattini invece di barre: `2021-22` (non `2021/22`)
- Esempi: `2021-22`, `2022-23`, `2023-24`

### Tipi di Competizione (tutto minuscolo)
- `campionato` - Campionato principale
- `apertura` - Torneo di Apertura
- `clausura` - Torneo di Clausura  
- `memorial` - Memorial
- `latrina` - Coppa Latrina

### Estensioni Supportate
- `.png` (consigliato per qualit√†)
- `.jpg` / `.jpeg`
- `.webp` (formato moderno, dimensioni ridotte)

## Esempi di Nomi File Validi

- `2021-22_campionato.png`
- `2022-23_apertura.jpg`
- `2023-24_clausura.png`
- `2021-22_memorial.jpeg`
- `2022-23_latrina.webp`
- `2024-25_campionato.webp`

## Note

- I nomi dei file sono case-sensitive
- Assicurarsi che non ci siano spazi nei nomi dei file
- Le immagini verranno caricate automaticamente nella pagina delle competizioni corrispondenti
